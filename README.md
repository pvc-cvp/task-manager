# 📋 任务管理器 Task Manager

一个简洁美观的在线任务管理工具，帮助你高效管理日常待办事项。

## 🌐 在线访问

👉 [点击这里打开](https://pvc-cvp.github.io/task-manager/)

## ✨ 功能特点

- ✅ 添加 / 删除任务
- ✅ 标记任务完成
- ✅ 数据本地存储，刷新不丢失
- ✅ 简洁美观的界面设计
- ✅ 支持手机和电脑访问

## 🛠️ 技术栈

- HTML
- CSS
- JavaScript
- LocalStorage
